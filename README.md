# Simple Browser Clock

Notice: This was edited by me to add a proper hour field. Original author: https://github.com/dneuman/clock

This is a simple browser clock that is based on (read: basically a copy of) dneuman's code. I've modified this to add seconds, make the font white, and add Kindle support (specifically PW3 via a simple and very jank viewport width.)

To use, just point your browser to https://tylerwdoesstuff.github.io/clock/index.html.
